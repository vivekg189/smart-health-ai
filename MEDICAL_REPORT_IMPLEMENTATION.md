## üîß How It Works

### For Users:
1. User fills out disease prediction form
2. System analyzes and provides prediction
3. User clicks "Download Medical Report" button
4. Professional PDF report is generated and downloaded automatically
5. Report filename: `HealthAI_[Disease]_Assessment_[ReportID].pdf`

### For Developers:
```javascript
import { generateMedicalReport } from '../utils/reportGenerator';

// Simple usage in any form component
generateMedicalReport(
  'diabetes',  // Disease type
  formData,    // Input parameters
  result,      // Prediction results
  patientInfo  // Optional patient details
);
```
